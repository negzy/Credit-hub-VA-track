## Overview
This Command Center helps you stay organized during your 2-hour shifts, 3 days per week. Everything is designed to be simple, fast, and consistent.

## When there are multiple VAs

- **Same link, same dashboard** ‚Äî everyone uses the same URL. Your tasks and queues are on your own device (they don't sync).
- **Always select your name** ("I am: Elmo" / "Soc" / etc.) so the report says "Report from: [your name]". Emmanuel needs to see who did what.
- **One shared Doc for everyone** ‚Äî Emmanuel pastes the Doc link in "For Emmanuel" at the bottom. All VAs paste their end-of-shift report into that same Doc. Each report is labeled with your name.
- **Queues:** If you need one shared list (so no one double-replies), use one shared Google Sheet for Engagement + DM queues; everyone updates it. Use the dashboard for templates, tasks, and generating your report.

## Daily Workflow (2-Hour Shift)

### **Start of Shift (5 minutes)**
1. Open the dashboard (`dashboard.html`)
2. Review today's task list
3. Check the Engagement Queue for any urgent items (marked High priority)
4. Check the DM Queue for any follow-ups due today
5. Review Post Guidelines before posting anything

### **10 Minutes: Check Notifications & Tag Urgent**
- Go through all notifications in Skool
- Tag anything urgent in the Engagement Queue or DM Queue
- Mark priority as "High" for anything that needs immediate attention
- Update status to "In Progress" when you start working on it

### **30 Minutes: Reply to Comments (Engagement Queue)**
- Open the Engagement Queue table
- Start with High priority items, then Medium, then Low
- For each comment:
  1. Click the link to open the post/comment
  2. Read the full context
  3. Choose the appropriate template from the Templates section
  4. Customize the template (replace [Name], add personal touches)
  5. Copy and paste your reply
  6. Update the Status to "Done" in the queue
  7. Add any notes about the interaction

**Response Types:**
- **Answer**: Direct response to their question
- **Welcome**: New member greeting
- **Redirect**: Point them to Credit GPS or Start Here
- **Celebrate**: Acknowledge their win or progress
- **Ask follow-up**: Engage them with a question

### **30 Minutes: Welcome + Onboarding**
- Check for new members (look for introductions or "new here" posts)
- Send welcome messages using the Welcome template
- Answer questions about where to start
- Direct people to Start Here section and Credit GPS tool
- Update the Engagement Queue as you go

### **30 Minutes: Emmanuel Admin DMs + Follow-ups**
- Open the DM Queue
- Filter by "Follow-up date" = today
- For each DM:
  1. Read what they asked originally
  2. Check their Qualification stage
  3. Follow the rule: **ONE question only** if they show intent, then direct to next step
  4. Use appropriate template (DM Follow-up, Book a Call Redirect, etc.)
  5. Update Status to "Done" or "In Progress"
  6. Set next Follow-up date if needed
  7. Add notes about the conversation

**Qualification Stages:**
- **New**: Just reached out, need to understand what they want
- **Interested**: They've shown interest, ask ONE qualifying question
- **Needs clarity**: They're confused, clarify and redirect
- **Ready to book**: Send booking link
- **Not fit**: Use closing loop template

**Next Actions:**
- **Reply**: Send initial response
- **Ask 1 question**: ONE qualifying question only
- **Send link**: Send Credit GPS or booking link
- **Remind**: Follow-up reminder
- **Close loop**: Not a fit, polite closure

### **20 Minutes: Post Engagement Prompt**
- Check the community vibe (what's trending, what people are talking about)
- Either:
  - Post a new engagement prompt (use "Post Your Win Prompt" template)
  - OR bump an existing thread that's getting good engagement
- Add a relevant GIF to make it more engaging
- Check Post Guidelines before posting to ensure you're following rules

### **End of Shift (5 minutes): Update Metrics + Handoff**
1. **Update Quick Metrics:**
   - New Members: Count how many you welcomed
   - Comments Replied: Count items marked "Done" in Engagement Queue
   - DMs Sent: Count items marked "Done" in DM Queue
   - Calls Booked: Count any calls that were booked
   - Upgrades: Count any upgrades that happened

2. **Generate report for Emmanuel:**
   - Click **‚ÄúGenerate report for Emmanuel‚Äù** (in the handoff section).
   - Click **‚ÄúCopy report‚Äù** and paste the report into your shared Google Doc (or email/Slack).
   - Emmanuel uses this to see tasks done, queues, metrics, and your handoff note. Do this every shift.

3. **Write Handoff Note:**

## Weekly Cadence (3 Workdays)

### **Day 1 (Monday or First Workday)**
- Focus: Catch up on weekend activity
- Priority: High-priority items from Engagement Queue
- Special: Review all pending follow-ups and prioritize

### **Day 2 (Mid-Week)**
- Focus: Maintain engagement momentum
- Priority: New comments and DMs
- Special: Post engagement prompt to boost activity

### **Day 3 (Friday or Last Workday)**
- Focus: Close out the week strong
- Priority: All pending follow-ups before weekend
- Special: Update weekly metrics summary

## Using Templates

1. **Click on any template** in the Templates section
2. The template will be copied to your clipboard
3. **Paste it** into Skool or DM
4. **Customize it:**
   - Replace `[Name]` with their actual name
   - Replace `{{CREDIT_GPS_LINK}}` with the actual link
   - Replace `{{BOOKING_LINK}}` with the actual booking link
   - Add personal touches (emojis, specific details from their message)
   - Make it sound human, not robotic

**Important:** Templates are starting points. Always personalize them!

## Deleting queue items

- You can **delete** a row if you added it by mistake: click **Delete** on that row and confirm. Use this for duplicates or wrong entries. Data is saved automatically.

## Post Guidelines

**Before posting ANYTHING in any group:**
1. Check the Post Guidelines section in the dashboard
2. Read the rules for that specific group
3. Make sure your post follows:
   - Do's (what's encouraged)
   - Don'ts (what's not allowed)
4. If unsure, don't post - leave a note in handoff for Emmanuel

## Google Sheets Integration

### **Setting Up:**
1. Import `google-sheets-template.csv` into Google Sheets
2. You'll have three tabs: Engagement Queue, DM Queue, Quick Metrics
3. Update the dashboard manually OR copy/paste from sheets

### **Weekly Sync:**
- At end of week, copy data from dashboard to Google Sheets
- This creates a backup and allows Emmanuel to review trends

## Troubleshooting

**Dashboard not saving?**
- Make sure you're using a modern browser (Chrome, Firefox, Safari)
- Check browser console for errors (F12)

**Templates not loading?**
- Make sure `templates.txt` is in the same folder as `dashboard.html`
- You can edit templates.txt directly to add/change templates

**Need to reset daily tasks?**
- Click "Reset Daily Tasks" button
- This unchecks all tasks (useful if you want to start fresh)
